Sobre a Atividade

Ex Machina

Trabalho produzido pelos alunos do 2ºA Informática com o objetivo de desenvolver um site relatando toda a Sinopse do filme “Ex Machina” para qualquer público. Este site foi desenvolvido como forma de avalição bimestral da matéria de Filosofia ministrada para os alunos do 2°A informática IFRO Campus Vilhena. Para a produção/desempenho do trabalho, a turma foi dividida em dois grupos principais: Elaboração do Resumo e Desenvolvimento do Site.

Elaboração do Resumo

Para contextualizar, o filme é dividido por ATOS (7 atos), e, no filme, os atos são separados a cada dia que é passado no filme, onde, cada ato representa uma etapa diferente de Caleb na casa de Nathan. Desta forma, a turma foi separa em grupos, onde cada grupo ficou responsável para desenvolver um Ato especifico do filme, fazendo toda a elaboração da estrutura textual.

Desenvolvimento do Site

Utilizando toda a estrutura textual desenvolvida pelos grupos, foi desenvolvido o site utilizando as tecnologias: HTML, CSS e JavaScript, além do framework Lottie utilizado para desenvolver animações com a linguagem de programação JavaScript. Para o desenvolvimento do layout do site, incluindo o padrão de cores, a tipografia e a estrutura visual, foram consideradas os elementos e temas apresentados no filme, ou seja: TECNOLOGIA! Todo o layout e estrutura visual foram desenvolvidas pensando em elementos e cores que remetem a tecnologia.

    Estrutura da Página Inicial

  • Carrossel de Imagens: Apresentando com poucas informações todos os Atos.
  • Sinopse do Filme: Breve sinopse apresentando o filme.
  • Atos do Filme: Todos os dados específicos de cada ato que ocorre no filme.

    Estrutura de Páginas

  • Ex Machina: Página responsável pelas informações do filme.
  • Sobre a atividade: Este Texto.
  • Envolvidos: Todos os integrantes responsáveis por cada ato.
